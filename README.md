# FindPro Open MVP

這是一個免後端即可開啟的專家媒合平台原型，適合先驗證流程與商業模式。

## 已包含功能

- 顧客 AI 問答：先拆解問題、推薦服務分類與發布需求前的準備事項。
- 顧客發布需求：填寫分類、地區、預算、時間、聯絡方式與細節。
- 專家註冊：建立服務分類、服務地區、價格與聯絡方式。
- 找專家：依分類與關鍵字搜尋專家。
- 需求看板：公開顧客需求並推薦匹配專家。
- 名單匯出：將顧客需求與專家名單匯出 JSON。

## 使用方式

直接用瀏覽器開啟 `index.html`。

## 下一階段建議

- 串接正式登入：顧客、專家、管理員三種角色。
- 串接資料庫：需求、專家、分類、地區、聯絡紀錄。
- 串接 OpenAI API：由 AI 追問需求細節、產生分類與專家匹配分數。
- 加入審核機制：專家身分、作品、評價與檢舉。
- 加入通知：Email、LINE Notify 或簡訊。
- 加入後台：名單管理、分類管理、SEO 頁面與營運儀表板。
